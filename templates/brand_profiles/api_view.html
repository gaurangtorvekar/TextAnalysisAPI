Text Analysis API

Suppose your document is a sentence - "This is an awesome movie!", send your document to 'www.gaurangtorvekar.com/api/analyse/'

Your input data should be in json format like so -

{"json":{"text":"This is an awesome movie!"}}


The response is in the following format

 {'sentiments':
         {'positive':1, 'negative':0},
   'frequent_words':[],
   'pos_tags':
         {'adjectives':[], 'verbs':[], 'nouns':[]}
 }
